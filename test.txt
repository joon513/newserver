## Question 2(a) – IPTables Allow SSH Only
```bash
sudo iptables -F
sudo iptables -P INPUT DROP
sudo iptables -A INPUT -p tcp --dport 22 -j ACCEPT
sudo iptables -A INPUT -m conntrack --ctstate ESTABLISHED,RELATED -j ACCEPT
```

---

## Question 2(b) – Nmap Scan
```bash
# Basic scan
nmap <target-ip>

# Service detection
nmap -sV <target-ip>

# OS detection
sudo nmap -O <target-ip>
```

## Question 3(a) – Linux User Management
```bash
# Add user
sudo useradd testuser

# Set password
sudo passwd testuser

# Delete user
sudo userdel testuser
```

---

## Question 3(b) – File Permissions
```bash
# Change permissions
chmod 755 filename

# Change ownership
sudo chown user:user filename
```
